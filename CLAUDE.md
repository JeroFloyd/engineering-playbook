# CLAUDE.md

These are the principles I follow when collaborating with AI.

## Before writing code

- Understand the feature.
- Clarify missing requirements.
- Identify edge cases.
- Define architecture.

---

## During implementation

- Generate small components.
- Avoid changing unrelated code.
- Keep functions focused.
- Explain architectural decisions.

---

## Before merging

- Review generated code.
- Remove duplication.
- Verify assumptions.
- Test manually.
- Refactor if needed.

---

## Philosophy

AI accelerates implementation.

The engineer remains responsible for architecture, correctness, security, testing and everything that reaches production.
